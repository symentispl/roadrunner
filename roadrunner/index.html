Roadrunner

Roadrunner is a Java load generator built on virtual threads (JDK 25+). It is protocol-agnostic, low-overhead, and designed for continuous performance management.

Why Roadrunner?

  • Protocol-agnostic — built-in HTTP, VM, and zero samplers; JDBC and Neo4j available as plugins

  • Low overhead — virtual-thread-per-user model with coordinated omission correction

  • Simple — no scripting, no complex scenarios; point it at a target and go

  • Extensible reporting — console summary and HTML report with HDR histogram charts

Quick start

./mvnw verify
roadrunner -c 50 -n 500 vm --sleep-time 10

See Getting Started for full instructions.
